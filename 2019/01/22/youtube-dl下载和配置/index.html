下载视频工具youtube-dl的安装与配置
发布于：2019-01-22 14:26:22 标签：/ windows / download / youtube-dl / 访问：

一个下载视频的软件，安装后直接在cmd使用。为了从BBC iplayer下视频发现的。
记一下安装过程：

  1. 安装python，勾选添加到PATH和安装pip。安装后可能需要重启才能使用pip。
  2. 安装ffmpeg，直接从官网下载，然后解压将bin目录添加进PATH即可。

  3. 使用pip安装youtube-dl

  4. 完成啦，使用指南
