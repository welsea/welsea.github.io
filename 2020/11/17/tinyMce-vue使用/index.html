tinyMce-vue使用
发布于：2020-11-17 15:48:22 标签：/ vue / 访问：

安装

安装tinyMce-vue和tinyMce。
因为vue是2.x所以tinymce-vue需要安装3.x版本。
这里是tinymce ^5.5.1，tinymce/tinymce-vue ^3.2.6




准备工作

需要把node_modules里/tinymce/skins复制出来，粘贴在static/tinymce中 (文件夹自己新建）。

需要语言包的去官网下载，解压后同样放在static中。



封装组件

因为在项目里需要用到的地方不止一个，封装组件比较方便。
注意前面的引入和其他教程不同：

  • 引入import 'tinymce/icons/default'，不然会报错 icon.js:1 Uncaught SyntaxError: Unexpected token '<'。
  • 引入 import '../../static/tinymce/zh_CN.js'，不然会报错 zh-CN.js:1 Uncaught SyntaxError: Unexpected token '<'。

这样就完成了一个基础的tinyMce使用。



增加plugins

按照官方文档，增加plugins需要引入。
